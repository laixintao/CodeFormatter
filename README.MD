# Code Formatter
# 代码格式转换器

这是一个用于转换代码格式的工具，此工具不会影响代码的时机内容，只会改变代码的阅读性。

## 使用方法
目前的版本支持在windows的cmd上运行（因为要接收参数）。

 1. 将编译好的exe执行文件拷贝要处理的文件所在文件夹。
 2. 控制台切换到此文件夹。
 3. 程序名 处理文件名。
 4. 生成的“dest.txt”就是目标文件。
